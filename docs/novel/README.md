# 小说式教程：从零到精通（序）

这是一部用故事带你看懂「强化学习 + 世界模型 + MuJoCo」的小说式教程。你不需要任何先验知识，只要跟着章节一步一步读与做，最后你会：
- 理解智能体如何和环境互动、如何学习策略与价值；
- 掌握 PPO 的训练细节与 GAE 的作用；
- 认识轻量世界模型（Encoder + Dynamics）如何帮助学习；
- 能从 `train.py` 出发，读懂项目每一步代码的意图与实现。

阅读建议：
- 每章都有“剧情推进”和“小练习”，边读边动手，效果最好。
- 章节内会链接到项目中的具体代码与文档，遇到不懂的术语，先略过，跟着情节走。
